# Seoly Digital — SEO & Digital Marketing Agency Website

![Seoly](https://img.shields.io/badge/Status-Live-brightgreen) ![HTML](https://img.shields.io/badge/HTML5-E34F26?logo=html5&logoColor=white) ![CSS](https://img.shields.io/badge/CSS3-1572B6?logo=css3&logoColor=white) ![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?logo=javascript&logoColor=black) ![Vercel](https://img.shields.io/badge/Deployed%20on-Vercel-black?logo=vercel)

> A fully responsive, production-grade digital marketing agency website with AI-powered live chat, client dashboard, and auth system — built with pure HTML, CSS & JavaScript.

🔗 **Live Site:** [seoly.co.uk](https://seoly.co.uk)

---

## ✨ Features

- **AI Chatbot** — Powered by Claude (Anthropic API) for real-time 24/7 visitor support
- **Auth System** — Sign In / Sign Up with email + Google + Facebook OAuth (modal UI)
- **Client Dashboard** — Post-login portal with SEO reports, keyword tracker, profile, settings, billing
- **Animated UI** — Custom cursor, scroll reveal animations, glowing orbs, grid backgrounds
- **SEO Optimised** — Schema.org LocalBusiness markup, meta tags, OG tags, keyword-rich content
- **Fully Responsive** — Mobile-first design across all screen sizes
- **Pure Vanilla** — Zero frameworks, zero dependencies — just HTML, CSS & JS

## 🛠 Tech Stack

## 🚀 Deploy Your Own

1. Fork this repository
2. Go to [vercel.com](https://vercel.com) → New Project → Import from GitHub
3. Select this repo → Deploy
4. Add your custom domain in Vercel settings
